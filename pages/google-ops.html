LocalLift Studio

Google Ops

Correo profesional, Google Business Profile, reseñas y reservas gestionadas.

Flujo operativo para clientes que contratan LocalLift: dejamos su presencia Google ordenada, medible y conectada a la web.

Abrir brief

1. Correo profesional

Se crea en Google Workspace, no como Gmail personal. Requiere dominio del cliente, alta de Workspace, facturacion y permisos de administrador.

  • Correo recomendado: info@, reservas@, hola@ o citas@.
  • Activar cambio de contraseña en primer inicio.
  • Configurar firma, recuperacion y reenvios si procede.

2. Google Business Profile

El cliente debe invitarnos como manager o propietario autorizado. Desde ahi se revisan datos, enlaces, horarios, web, categorias y acciones.

  • Sincronizar telefono, direccion, web y horarios.
  • Añadir enlace de reservas, citas, pedidos o contacto.
  • Usar Place ID, Maps URL y review URL dentro de LocalLift.

3. Reservas y citas

El primer nivel usa URL de reserva/WhatsApp. El nivel avanzado conecta Google Calendar con OAuth del negocio.

  • Comprobar disponibilidad con FreeBusy.
  • Crear eventos en Calendar con datos del lead.
  • Definir duracion, buffers, servicios y confirmacion manual.

4. Reseñas y reputacion

LocalLift puede preparar mensajes para pedir reseñas, listar reseñas y redactar respuestas. Publicar respuestas debe requerir aprobacion humana.

  • Enviar enlace de reseña tras visita/reserva.
  • Detectar reseñas sin respuesta.
  • Preparar borradores con tono del negocio.

5. Permisos necesarios

Para operar bien necesitamos accesos separados por cliente, nunca claves dentro del HTML exportado.

  • Admin de Google Workspace para crear usuarios.
  • OAuth de Business Profile para perfil, ubicaciones y reseñas.
  • OAuth de Calendar para disponibilidad y reservas.
  • API key de Maps/Places en backend para enriquecer datos.

6. Entregable mensual

Esto justifica mantenimiento recurrente: revisar Google, responder reseñas, mejorar reservas y reportar resultados.

  • Leads, clics, llamadas, mapas y conversaciones.
  • Reseñas nuevas, pendientes y respondidas.
  • Cambios aplicados en web, Google y reservas.
