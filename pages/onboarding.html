01

Correo profesional

Se crea en Google Workspace, no como Gmail personal. Requiere el dominio del cliente, alta de Workspace, facturacion y permisos de administrador.

  • Definir cuentas como info@, reservas@, hola@ o citas@.
  • Configurar firma, recuperacion, reenvios y cambio de contrasena.
02

Google Business Profile

El cliente nos invita como manager o propietario autorizado para revisar datos, horarios, categorias, web y acciones.

  • Sincronizar telefono, direccion, web y horarios.
  • Conectar Place ID, Maps URL, resenas y enlaces de reserva.
03

Reservas y citas

El nivel inicial usa una URL de reserva o WhatsApp. El nivel avanzado conecta Google Calendar mediante OAuth del negocio.

  • Comprobar disponibilidad y crear eventos con los datos del lead.
  • Definir duracion, margenes, servicios y confirmacion.
04

Resenas y reputacion

LocalLift prepara mensajes para solicitar resenas y borradores de respuesta. La publicacion final requiere aprobacion humana.

  • Enviar el enlace de resena despues de una visita o reserva.
  • Detectar resenas pendientes y responder con el tono del negocio.
05

Permisos necesarios

Los accesos se gestionan por cliente y nunca se incluyen claves dentro de la web exportada.

  • Admin de Workspace y OAuth de Business Profile y Calendar.
  • API de Maps o Places configurada de forma segura en backend.
06

Seguimiento mensual

El mantenimiento mide resultados y mantiene coordinadas la web, la presencia local y las reservas.

  • Leads, clics, llamadas, mapas, conversaciones y resenas.
  • Cambios aplicados en la web, Google y el sistema de reservas.
