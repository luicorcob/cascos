# Paquete para revision legal y fiscal

Objetivo: obtener una respuesta util de asesoria y gestoria en una sola ronda,
sin enviar documentos aislados ni preguntas ambiguas.

## Documentos a enviar

- `docs/ventas/OFERTA_LOCAL_LIFT_RESERVAS.md`
- `docs/ventas/CONDICIONES_SERVICIO_FUNDADORES_BORRADOR.md`
- `docs/ventas/FACTURA_ANTICIPO_PLANTILLA.md`
- `docs/ventas/INSTRUCCIONES_COBRO_DEPOSITO.md`
- `pages/privacy-demo.html`, como ejemplo que debe sustituirse para clientes
  reales.

## Datos que debe aportar el prestador

- Nombre o razon social, NIF, domicilio, email y telefono.
- Situacion fiscal y fecha prevista de inicio de actividad.
- Si el cliente objetivo contrata siempre como empresa o profesional.
- Proveedor de hosting, dominio, correo, copias y formularios.
- Quien accede a contactos y reservas y durante cuanto tiempo se conservan.
- Medio previsto para cobrar deposito, pago final y mensualidad.

## Preguntas para asesoria legal

1. Que texto de privacidad y aviso legal necesita la web de LocalLift.
2. Que texto necesita cada web de cliente y quien figura como responsable.
3. Si LocalLift actua como encargado de tratamiento y que acuerdo hace falta.
4. Como regular cancelacion, devolucion, retrasos y falta de materiales.
5. Que limites de responsabilidad y jurisdiccion son adecuados.
6. Como registrar de forma valida la aceptacion de propuesta y condiciones.

## Preguntas para gestoria

1. Como debe facturarse el anticipo de 245 EUR y el pago final.
2. IVA, retenciones y menciones obligatorias aplicables.
3. Serie, numeracion y fecha de expedicion.
4. Como reflejar el anticipo en la factura final.
5. Requisitos de software, QR y factura electronica aplicables.
6. Como documentar el mantenimiento mensual.

## Evidencia para cerrar Fase 0

- [ ] Respuesta legal recibida y cambios incorporados.
- [ ] Texto de privacidad de demo sustituido o aprobado para su uso.
- [ ] Condiciones de servicio aprobadas.
- [ ] Respuesta fiscal recibida y plantilla de factura corregida.
- [ ] Medio de cobro aprobado y verificado.
