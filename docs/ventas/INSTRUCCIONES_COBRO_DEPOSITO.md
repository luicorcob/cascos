## Metodo inicial recomendado

Usar transferencia bancaria para los primeros clientes fundadores. Evita
integrar pagos antes de validar la oferta y permite identificar el anticipo con
una referencia clara.

Datos que debe completar el prestador:

```text
Titular: [TITULAR]
IBAN: [IBAN]
Entidad: [ENTIDAD]
Importe del deposito: 245 EUR antes de impuestos cuando correspondan
Concepto: LOCALLIFT [NEGOCIO] [FECHA]
Email para enviar justificante: [EMAIL]
```

## Mensaje al cliente

> Para reservar la fecha de inicio, realiza el deposito indicado en la
> propuesta usando los datos adjuntos. Incluye como concepto `LOCALLIFT
> [NEGOCIO] [FECHA]` y envia el justificante a [EMAIL]. Confirmaremos por
> escrito la recepcion y el inicio del plazo cuando tambien tengamos los
> materiales minimos.

## Validacion obligatoria

- [ ] Los datos bancarios pertenecen al prestador verificado.
- [ ] El importe coincide con las condiciones revisadas.
- [ ] La gestoria ha confirmado como documentar y facturar el anticipo.
- [ ] Se ha realizado una transferencia de prueba o verificacion equivalente.
- [ ] Existe un registro de propuesta, aceptacion, pago y justificante.
- [ ] Se ha definido como cobrar el pago final y el mantenimiento mensual.

No enviar instrucciones de cobro ni aceptar pagos hasta completar esta lista.
